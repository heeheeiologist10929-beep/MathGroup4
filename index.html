Welcome to Exponential Functions

Exponential functions are mathematical functions of the form:

f(x) = bx

where b is the base and x is the exponent.

They are used to model many real-world phenomena such as population growth, radioactive decay, and compound interest.

Examples:

  • Population growth: doubling every 10 years.
  • Radioactive decay: half-life of substances.
  • Compound interest in finance.

Exponential Functions, Equations, and Inequalities

An exponential function is a function of the form:

f(x) = bx

where b is any number such that b > 0 and b ≠ 1. In this case, b is called the base and x can be any real number.

Exponential equations are equations in which variables occur as exponents.

Examples:

  • 2x+1 = 8
  • 32x-1 = 3x+1
  • 42x = 25-x
  • 10x = 100x-2

Likewise, exponential inequalities are inequalities in which variables occur as exponents.

Examples:

  • 2x+1 < 8
  • 32x-1 ≤ 3x+1
  • 42x ≥ 25-x
  • 10x > 100x-2

Solving Exponential Equations and Inequalities

Step 1. Rewrite both sides with the same base that is nonzero.

Step 2. Equate the exponents on both sides of the equation.
(This is the application of Law 11, that is, if ax = ay, then x = y.)

Step 3. Solve for x.

Examples:

  1. Given: 3x = 81
  2. Rewrite 81 as 34
  3. Set exponents equal: x = 4
  4. Solution: x = 4
  1. Given: 2x+1 = 16
  2. Rewrite 16 as 24
  3. Set exponents equal: x + 1 = 4
  4. Solve for x: x = 3

Example: Evaluating g(x) = 3-x

Consider the function g(x) = 3-x. Let's evaluate it at some points:

  • g(-3) = 3-(-3) = 33 = 27
  • g(0) = 3-0 = 30 = 1
  • g(3) = 3-3 = 1 / 33 = 1 / 27 ≈ 0.037

Plotting Exponential Functions

Follow these steps to plot exponential functions:

  1. Construct a T-table for values in the domain.
  2. Plot the points from the T-table.
  3. Trace the curve connecting the points.
Example:

Plot the functions below:

  • f(x) = 2x
  • g(x) = 2-x
  • h(x) = -2x

Functions f(x) = ax and g(x) = -ax

Consider the functions f(x) = ax and g(x) = -ax for some positive number a.

+ +
Function f(x) g(x)
Domain Real numbers Real numbers
Range Positive numbers Negative numbers
Asymptote x-axis x-axis
+

Note that f and g both have no zero. Otherwise, for each function, it leads to a contradiction as shown below.

+

Example:

+

Let a > 0 and a ≠ 1. Suppose f(x) = ax has a zero such that there exists a real number x such that

+
f(x) = 0
+
ax = 0
+

But since a > 0, then for all x, ax > 0. This contradicts the equation above that ax = 0. Therefore, f(x) = ax has no zero.
